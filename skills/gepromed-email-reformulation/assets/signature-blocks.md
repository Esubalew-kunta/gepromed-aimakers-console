# GEPROMED signature blocks

Standard text signature blocks to append when the user provides a name and
designation (or asks for a signature). Keep them clean and institutional. For a
rendered PNG signature image, use the separate `gepromed-signature-generator`
skill; the logo asset here (`gepromed-logo.png`) is bundled for reference.

## FR — standard
```
Bien cordialement,

[Prénom Nom]
[Fonction]
GEPROMED — The medical device hub for patient safety
[téléphone]  ·  [email]  ·  gepromed.com
```

## EN — standard
```
Kind regards,

[First name Last name]
[Designation]
GEPROMED — The medical device hub for patient safety
[phone]  ·  [email]  ·  gepromed.com
```

## Minimal (internal / short)
```
[Prénom Nom] — GEPROMED
```

## Rules
- Only include phone / email / website if the user provides them; otherwise leave
  `[bracketed placeholders]` or omit.
- Do not add taglines, quotes, emojis, social icons, or legal disclaimers unless
  asked.
- Department line (optional): "GEPROMED — Test Platform / René Kieny Education
  Center / Clinical Research / Explant Analysis" matching the sender's pillar.
- Brand colours apply only to rendered/HTML signatures: primary blue `#007AC2`,
  text `#1F2A33`, muted `#5F6B73`. Do not overuse orange.
