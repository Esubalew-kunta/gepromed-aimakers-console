# Offre d'emploi / Job posting — structure (STRUCTURE + placeholders)

A **structural template**, not legal text. It tells the skill *what a GEPROMED job
posting contains* and *what must be filled in*. Every square-bracketed item is a
**placeholder the RAF (and hiring manager) confirms** — the skill never replaces a
bracket with an invented value (salary, CCN classification, contract terms).

Bilingual labels FR / EN. Mirror the requested language in output.

> ZERO-INVENTION: never invent a salary, a coefficient/classification from the CCN,
> a contract duration, a trial period, or a benefit. Bracket each and name the
> source the RAF checks (the applicable CCN, the company's HR policy).

---

## Recommended sections

### 1. Titre du poste / Job title
`[intitulé — H/F ou formulation inclusive à confirmer]`. Clear, non-discriminatory.

### 2. À propos de GEPROMED / About GEPROMED
2–4 sentences of mission and credibility, from `brand-guidelines.md` (non-profit
medical-device hub for patient safety; four pillars; ISO/Qualiopi). Mission, not
sales. Proof points only if true.

### 3. Le poste / The role
- **Rattachement / Reports to:** `[équipe / pilier — e.g. Test, Education, Clinical Research, Explant Analysis]`.
- **Missions principales / Key responsibilities:** 4–8 concrete bullets `[à préciser avec le manager]`.
- **Type de contrat / Contract type:** `[CDI / CDD / … — à confirmer]`; if CDD, `[motif et durée — à confirmer par le RAF]`.
- **Temps de travail / Working time:** `[35h / 39h / autre — à confirmer]`.
- **Lieu / Location:** `[Strasbourg / autre]`; télétravail `[à confirmer]`.
- **Date de prise de poste / Start date:** `[à confirmer]`.

### 4. Profil recherché / Candidate profile
- **Formation / Education:** `[diplôme / niveau]`.
- **Expérience / Experience:** `[années / domaine]`.
- **Compétences / Skills:** technical + soft `[à préciser]`.
- **Langues / Languages:** `[FR / EN — niveau]`.
> Keep criteria job-related and non-discriminatory. No age, sex, origin, health,
> or family-situation criteria. Flag anything doubtful for the RAF's equality check.

### 5. Conditions / What we offer
- **Rémunération / Salary:** `[fourchette / classification CCN — à confirmer par le RAF]` (range only; never an invented figure).
- **Avantages / Benefits:** `[tickets restaurant, mutuelle, etc. — à confirmer]`.
- **Classification conventionnelle:** `[coefficient / position selon la CCN — à confirmer par le RAF]`.

### 6. Candidature / How to apply
- **Pièces / Documents:** CV + `[lettre de motivation? — à confirmer]`.
- **Contact / Process:** `[adresse / portail de candidature — à confirmer]`.
- **Mention RGPD / Data note:** a short line that applications are processed for
  recruitment; for the full notice, link to the privacy policy `[lien — à confirmer]`.
  (Coordinate with the RGPD skill / DPO for the exact wording.)

### 7. Égalité & inclusion / Equality & inclusion
A short statement that GEPROMED is an equal-opportunity employer and welcomes all
qualified applicants `[exact wording — à confirmer par le RAF]`.

---

## Rules for every job posting
- Replace a placeholder **only** with a value the user supplied or `MEMORY.md`
  confirmed; otherwise keep the bracket and list it under "Sources to verify".
- Never invent a salary, CCN coefficient, contract duration, trial period, or benefit.
- Keep all criteria job-related and non-discriminatory; flag doubts for the RAF.
- End the output with the VALIDATION FLAG (RAF) and the HR-legal disclaimer.
