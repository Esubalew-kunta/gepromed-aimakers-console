# SEO & GEO guidance (GEPROMED website content)

How to make a GEPROMED page rank in classic search (**SEO**) *and* be quotable by
generative engines (**GEO** — Generative Engine Optimization: Google AI Overviews,
ChatGPT, Perplexity, etc.). Both reward the same thing GEPROMED already wants:
clear, factual, well-structured, authoritative copy. Never trade clarity or
honesty for keywords.

## 1. Keyword strategy (per page)
- **One primary keyword/query** per page (the search intent the page answers).
- **2–3 secondary keywords** that support it (synonyms, sub-topics, audience terms).
- Place the **primary keyword** in: the `H1`, the **first 1–2 sentences** of the
  intro, the `meta title`, the `meta description`, and at least one `H2`.
- Use secondaries naturally in H2s and body — **never stuff**. If a sentence reads
  worse with the keyword, drop the keyword.
- Prefer the user's real language (e.g. "vascular explant analysis",
  "formation chirurgie vasculaire", "biomaterial testing") over internal jargon.

## 2. Meta title (the SERP/clickable title)
- **≤60 characters** (truncates beyond ~60). Front-load the primary keyword.
- End with the brand: `… | GEPROMED`.
- No hype, no ALL-CAPS, no clickbait. One title per page; unique across the site.
- Pattern: `<Primary keyword / capability> | GEPROMED`
  - EN: `Vascular Explant Analysis | GEPROMED`
  - FR: `Analyse d'explants vasculaires | GEPROMED`

## 3. Meta description
- **120–158 characters** (Google truncates ~155–160).
- One sentence or two: **what the page offers + the value + a soft action**, with
  the primary keyword used naturally and at least one real proof point when it fits.
- Not a keyword list; it is ad copy that must also be true.
  - EN: `Independent vascular explant analysis from the world's largest explant database. Helping manufacturers and clinicians improve device safety.`
  - FR: `Analyse indépendante d'explants vasculaires, adossée à la plus grande base d'explants au monde. Au service de la sécurité des dispositifs.`

## 4. Heading hierarchy (on-page SEO + accessibility)
- **Exactly one `H1`**, carrying the primary keyword and the page promise.
- `H2`s segment the page by sub-topic; `H3`s nest under H2s. **No skipped levels.**
- Headings are descriptive and scannable — a reader skimming only the headings
  should understand the page. Avoid clever/empty headings ("The magic happens").
- Put a relevant keyword in headings where natural, not in every one.

## 5. GEO — being the quotable answer
Generative engines extract and cite **self-contained, factual passages**. To be
the answer:
- **Answer the core question in the first 1–2 sentences** of the intro, plainly:
  *"Explant analysis is the examination of medical devices removed from the body
  to understand failure and improve safety. GEPROMED runs it independently for
  cardiovascular devices."* (Definition + who + why, in one quotable block.)
- Write **standalone passages**: each section should make sense if quoted alone,
  without "as mentioned above".
- Use **specific, attributable facts** (standards, methods, real proof points).
  Engines favor verifiable specificity over vague claims — which suits GEPROMED.
- Prefer **structured answers**: short definitions, ordered steps, labelled lists.
  A clear "How it works" with numbered steps is highly extractable.
- **Entity clarity**: name GEPROMED, the pillar, and the device class explicitly,
  so engines associate the page with the right entity and topic.
- **Never optimize with invented facts.** A fabricated number that gets cited is a
  credibility and compliance failure. Bracket unconfirmed claims.

## 6. Other on-page signals
- **One clear primary CTA** per page, specific and action-led.
- **Internal links** between related pages (homepage ↔ pillars ↔ authority ↔
  training) — suggest them in `Notes:` for Comms to wire in Ibexa.
- **Alt text** for images: describe the image factually (suggest in `Notes:`).
- **Readability**: short paragraphs, bullets for lists, active voice — this serves
  SEO, GEO, accessibility, and the brand's clarity goal at once.
- **Bilingual parity**: FR and EN versions should be equivalent in substance and
  each have their own correct meta title/description and keywords (not a literal
  translation of the other's keywords).

## 7. SEO/GEO checklist (run before returning)
- [ ] One primary keyword chosen; 2–3 secondaries.
- [ ] Primary keyword in H1 + intro (first 1–2 sentences) + meta title + meta description.
- [ ] Meta title ≤60 chars, ends `| GEPROMED`, unique.
- [ ] Meta description 120–158 chars, true, with a real proof point where it fits.
- [ ] Exactly one H1; logical H2/H3; descriptive headings.
- [ ] Intro is a self-contained, GEO-quotable answer.
- [ ] At least one extractable structured passage (definition / steps / labelled list).
- [ ] Specific CTA with a destination.
- [ ] No invented facts; unconfirmed claims bracketed.
- [ ] Internal-link and alt-text suggestions noted for Comms.
